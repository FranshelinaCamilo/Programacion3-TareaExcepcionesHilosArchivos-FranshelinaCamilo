## Guía de Uso

Para interactuar con el sistema y probar sus funcionalidades, siga esta guía:

### Ejercicio 1 - Simulador de Descargas

Simula la descarga simultánea de tres archivos utilizando hilos independientes. Cada archivo muestra su progreso hasta completar la descarga al 100%.

### Ejercicio 2 - Carrera de Corredores

Simula una carrera entre cuatro corredores utilizando hilos. Cada corredor avanza desde el metro 1 hasta el metro 10 y al finalizar se muestra el ganador de la carrera junto con el estado inicial y final de los hilos.

### Ejercicio 3 - Validador de Edad

Solicita el nombre y la edad de una persona. Si la edad ingresada es menor de 18 años, se lanza una excepción personalizada llamada EdadInvalidaException y se muestra un mensaje de error.

### Ejercicio 4 - Calculadora Segura

Calculadora con las siguientes opciones:

* Sumar
* Restar
* Multiplicar
* Dividir
* Salir

El menú se repite hasta que el usuario seleccione la opción Salir. El programa maneja errores por división entre cero y por entradas inválidas cuando se esperan números.

### Ejercicio 5 - Registro de Actividades

Permite registrar actividades en un archivo de texto llamado actividades.txt.

Opciones del menú

**1. Agregar actividad**

Solicita una actividad y la guarda en el archivo sin eliminar las actividades previamente registradas.

**2. Mostrar actividades**

Lee el archivo actividades.txt y muestra todas las actividades almacenadas.

**3. Salir**

Finaliza la ejecución del programa.

El menú se repite hasta que el usuario seleccione la opción Salir.
